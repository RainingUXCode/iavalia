# IAvalia - Protótipo de Interface

Este é um protótipo de interface para o sistema **IAvalia**, uma plataforma de correção inteligente desenvolvida pela Secretaria de Estado da Educação da Paraíba (SEE-PB).

## 📋 Sobre o Projeto

O IAvalia é um sistema de correção automatizada que utiliza inteligência artificial para avaliar provas e atividades escolares, facilitando o trabalho dos professores e proporcionando feedback mais rápido aos alunos.

Este protótipo apresenta as telas principais da aplicação, incluindo:

- **Página Inicial (Landing Page)**: Apresentação do sistema com seções de recursos, como funciona, footer institucional

## 🚀 Tecnologias Utilizadas

- **React 19** - Framework JavaScript
- **TypeScript** - Tipagem estática
- **Vite** - Build tool e dev server
- **Tailwind CSS** - Framework de estilos
- **Shadcn/ui** - Componentes de UI
- **Lucide React** - Ícones

## 📦 Instalação e Execução

### Pré-requisitos

- Node.js (versão 18 ou superior)
- npm ou pnpm

### Passos para instalação

1. **Clone o repositório** (se aplicável) ou navegue até a pasta do projeto:

   ```bash
   cd vite-project
   ```

2. **Instale as dependências**:

   ```bash
   npm install
   # ou
   pnpm install
   ```

3. **Execute o servidor de desenvolvimento**:

   ```bash
   npm run dev
   # ou
   pnpm dev
   ```

4. **Abra no navegador**:
   - A aplicação estará disponível em: `http://localhost:5173`
